Our Privacy Policy

We only collect anonym data provided by Google Analytics for visitor count and visitor information, such as:

  • Browser Type
  • Connection Type
  • Language
  • Location
  • Pages viewed

This information is used to improve the site. If you wish to opt out of this please set the Do Not Track header, you can find instructions on how to do this along with more information on Do Not Track.

If you wish to see what data our model uses, you can go on the dev page and export it. You will see that there's only numerical (vectors of the face and the hand) and boolean data (label you set).
